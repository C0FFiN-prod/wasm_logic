# Логический симулятор (JS + Canvas)

Минимальный прототип симулятора логических схем с возможностью добавления элементов (AND, OR, NOT, XOR, T-Flop, входы/выходы), соединения их проводами, запуска симуляции и сохранения/загрузки схем.

## Запуск

- `npm run dev` - чтобы запустить в режиме разработчика для просмотра изменений на лету.
- `npm run build` - чтобы собрать проект. Результаты сборки в ./dist
- Три точки в правом верхнем углу поля - меню с переключением темы и руководством пользователя
