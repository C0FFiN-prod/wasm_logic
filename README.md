# Логический симулятор (JS + Canvas)

Минимальный прототип симулятора логических схем с возможностью добавления элементов (AND, OR, NOT, XOR, T-Flop, входы/выходы), соединения их проводами, запуска симуляции и сохранения/загрузки схем.

## Запуск

1. Откройте `index.html` в браузере (Chrome/Firefox).
2. Используйте панель инструментов для добавления элементов.
3. Соединяйте элементы: ПКМ по элементу — режим соединения, затем клик по другому элементу.
4. Двойной клик по входу — переключение значения.
5. Кнопки "Старт/Стоп" — запуск/остановка симуляции.
6. "Сохранить" — экспорт схемы в файл, "Загрузить" — импорт из файла.

## Структура
- `logic.js` — ядро симуляции
- `main.js` — UI и взаимодействие
- `index.html` — страница
- `style.css` — стили
